# Presupuesto PC Económica para Oficina y Multimedia

CPU (PROCESADOR)
### Opción elegida: **Intel Core i3-10105 (4C/8T, 4.4 GHz, UHD 630)**
**Justificación:**  
- Tiene **gráficos integrados Intel UHD 630**, lo que elimina la necesidad de una tarjeta gráfica.
- Ofrece **buen rendimiento** para tareas de oficina y multimedia.
- Bajo consumo energético y **precio accesible**.

**Precio aprox.:** 102 

---

PLACA BASE (MOTHERBOARD)
### Opción elegida: **Gigabyte H510M-H**
**Justificación:**  
- Compatible con el Intel Core i3-10105 (socket LGA 1200).
- **Soporta hasta 64 GB de RAM** en dual-channel.
- Tiene suficientes **puertos USB y conexiones esenciales**.
- **Precio bajo** en comparación con otras placas LGA 1200.

**Precio aprox.:** 69  

---

RAM
### Opción elegida: **Crucial 8GB DDR4 3200MHz**
**Características:**  
- Tipo: **DDR4**  
- Frecuencia: **3200MHz**  
- Módulo único de **8GB** (permite ampliar a futuro).  

**Precio aprox.:** 18  

---

ALMACENAMIENTO
### Opción elegida: **Kingston A400 240GB SSD**
**Justificación:**  
- **Velocidad de lectura:** 500MB/s (mucho más rápido que un HDD).  
- **Capacidad suficiente** para sistema operativo y programas básicos.  
- **Buena relación calidad-precio**.  

**Precio aprox.:** 30  

---

FUENTE DE ALIMENTACIÓN
### Opción elegida: **Fuente Alimentación Nox Urano VX 650W 80 Bronze 120MM PWM**
**Justificación:**  
- **Eficiencia 80 Plus Bronce**, mayor estabilidad y menor consumo.  
- **650W es más que suficiente** para esta configuración sin tarjeta gráfica.  
- Marca confiable y **precio asequible**.  

**Precio aprox.:** 48 

---

GABINETE
### Opción elegida: **Torre PC Aerocool CS106 Caja PC Micro ATX Negro**
**Justificación:**  
- **Compatibilidad** con la placa base Micro-ATX.
- **Diseño simple**

**Precio aprox.:** $19
